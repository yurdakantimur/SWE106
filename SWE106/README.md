🎵 Project Description

This project is a music education platform designed for people who are passionate about music and want to learn how to play an instrument.

🏠 Home Page

The home page includes our main images and short information about our courses.
Users can quickly understand what our platform offers and how it can help them.

📊 Survey Page

On the survey page, we have a system that helps users find the most suitable musical instrument for their personality.
It also shows where they should start based on their level and interests.

🎸 Courses Page

On the courses page, users can see different musical instruments and course levels.

Beginner Level:
If the user has no experience, we teach basic finger movements and fundamental skills.
Intermediate Level:
If the user knows basic techniques but cannot play well, they practice simple songs to improve their skills.
Advanced Level:
If the user can play easy songs and wants to improve more, they receive professional training and practice more difficult pieces.

ℹ️ About Us Page

On the About Us page, detailed information is provided about our course, how it works, and where it is located.
Users can learn more about our system and our learning environment.

❌We don’t have a database in our project, so adding items to the cart cannot be saved and there is no payment page. In addition, the “Login” button does not perform its real function.
